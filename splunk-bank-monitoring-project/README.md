# Splunk Bank Monitoring Project

## Overview  
This project shows how to monitor bank logs and analyze security using Splunk.

## Contents  
- **dashboard-screenshots/** contains 7 screenshots of the dashboard and search results  
- **spl-queries.md** includes all SPL queries used in this project  
- **final-report.pdf** is the complete report exported from Google Docs
